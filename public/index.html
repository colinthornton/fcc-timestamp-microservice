Timestamp Microservice

Input a time in the format "Month Date, Year" such as January 5, 2017, or a unix timestamp in the address bar.

https://colin-fcc-time.herokuapp.com/January 5, 2017
https://colin-fcc-time.herokuapp.com/151435345

If it's valid, you'll get a JSON with the following format.

{
"unix": Number,
"natural": String
}
